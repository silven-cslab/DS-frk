# Stack (Array Representation): Pop Operation

---

### Input: A stack STK represented as an array, and a `top` index tracking the number of elements.
### Output: The stack STK with the topmost item removed.

---

### Algorithm: POP(STK, top)

    1. if top < 1 then

        1.1. print "No elements in the stack! Can't pop!!"

        1.2. exit

    2. end if

    3. top := top - 1

    4. print "Successfully popped an element from the stack."

    5. stop
