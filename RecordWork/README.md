# Record Work (Experiments 1-3)

This folder establishes the complete record-work pattern for the first three Data Structures experiments.

## Structure
- `Exp*_*/Experiment.md` : Aim, Theory, Description
- `Exp*_*/Algos/*.md` : Algorithms in repository format
- `Exp*_*/Program/*.c` : Full C implementation
- `Exp*_*/Output/SampleOutput.txt` : Multiple sample test runs
- `Exp*_*/Docx/*.docx` : Individual formatted Word document
