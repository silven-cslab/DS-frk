# Experiment 3: Linked List Traversal and Manipulation

## Aim
To implement linked list traversal and manipulation operations such as counting, searching, updating, and finding the middle node.

## Theory
Traversal processes each node sequentially. Manipulation operations rely on traversal to access specific nodes. Counting gives list size, searching locates values, update modifies node data, and middle-node detection can be done using slow/fast pointers in one pass.

## Description
Create a menu-driven C program to:
1. Traverse and display the list.
2. Count nodes.
3. Search value and return position.
4. Update node value at a given position.
5. Find the middle node using two-pointer technique.
6. Handle empty list and invalid position input.
